# WebpageFIN
## Instructions to Use the Code

1. Open our FIN_home.html code with any browser (eg.Chrome) and you will be directed to the home page of FRIEND-IN-NEED (FIN).
2. From this page you can navigate to the Login and Sign Up pages.
3. It will direct you to the Login and Sign Up pages respectively. 
4. We have designed a demo page for the Discover tab.
5. Navigate to the Discover tab in the navigation bar.
6. Upon clicking Discover you will be taken to a page with various interests.
7. You can select your specific interest to join that particular event or group.
